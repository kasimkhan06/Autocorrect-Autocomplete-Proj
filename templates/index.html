Spell Checker with Autocomplete

This is a project for the course Speech & Natural Language Processing at Goa Engineering College

It is a web application that provides spell checking and autocomplete features.

The spell checker uses the Norvig's spell checker algorithm.

The autocomplete feature uses a trie data structure to provide suggestions for a given prefix.

Technologies Used
